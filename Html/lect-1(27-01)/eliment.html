Element Tag Example

1. Block-level Elements


:- Always start on a new line
:- Take full width of the page

Example


This is a div

This is a paragraph

This is a heading

Common block elements:


div
p
h1 to h6
section
header
footer

2. Inline Elements

:- Do not start on a new line
:- Take only required width

Example:

This is span Link Bold text

Common inline elements:

span
a
b
i
strong

3. Empty (Void) Elements

:- Do not have a closing tag
:- No content inside

Example:



Common empty elements:

br
hr
img
input
